Usage:
1. Download the compressed folder and extract it.
2. In OBS, go to "Tools" -> "Scripts" and choose the file named "obsnotify.lua".
3. You'll see the settings, allowing you to pick different sound files for start and stop actions for recording and streaming. 2 default sound files are provided and auto-selected.

For the original Windows-compatible version: https://obsproject.com/forum/resources/obs-notification-sound.809/
